# Kelompok 12 — Pariwisata Daerah Jawa Barat
## Struktur Folder
- data/raw/ : data mentah asli, read-only
- data/clean/ : data setelah preprocessing
- notebooks/ : pipeline OSEMN berurutan
- outputs/figures/ : semua visualisasi
- dashboard/ : dashboard interaktif

## Cara Menjalankan
1. Jalankan notebook secara berurutan: 01 → 02 → 03 → 04 → 05
2. Setiap notebook membaca dari folder sebelumnya

## Dependencies
pandas, matplotlib, seaborn, plotly, scikit-learn, jupyter
